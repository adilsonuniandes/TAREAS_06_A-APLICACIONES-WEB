# TAREAS_06_A-APLICACIONES-WEB

Repositorio académico correspondiente a la asignatura **Aplicaciones Web**,  
**sexto semestre – Universidad UNIANDES**.

En este repositorio se almacenarán las tareas desarrolladas a lo largo del semestre, organizadas por semana, aplicando conceptos de desarrollo web frontend y backend, así como el uso de **Docker** como base de la infraestructura de ejecución.

---

## Objetivo del repositorio

- Centralizar las tareas del sexto semestre de la materia **Aplicaciones Web**.
- Aplicar buenas prácticas de organización y versionamiento de código.
- Utilizar **Docker** como mecanismo estándar de infraestructura durante todas las semanas del semestre.
- Facilitar la revisión y evaluación de cada entrega semanal.

---

## Organización de las tareas

- Cada tarea se entrega en una carpeta correspondiente a la semana académica.
- Dentro de cada semana se agrupan los entregables según la tecnología utilizada.
- Cada semana incluye su propio archivo **README.md**, donde se detalla:
  - El objetivo de la tarea.
  - Las tecnologías empleadas.
  - Los pasos necesarios para ejecutar o revisar la solución.

---

## Uso de Docker

Durante todo el semestre se utilizará **Docker** como herramienta base para la ejecución de las tareas, permitiendo:

- Estandarizar el entorno de desarrollo y ejecución.
- Evitar dependencias específicas del sistema operativo.
- Facilitar la revisión técnica por parte del docente.

Las instrucciones de uso de Docker se detallan en el README de cada tarea semanal.

---

## Revisión de las entregas

Para revisar una tarea, se debe ingresar a la carpeta de la semana correspondiente y seguir las instrucciones descritas en su archivo **README.md**.
