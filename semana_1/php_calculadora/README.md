## Objetivo de la tarea
Aplicar conocimientos de:
- Instalación/uso de dependencias para ejecutar PHP.
- Sintaxis básica de PHP.
- Manejo de variables y condicionales.
- Integración de un script PHP con HTML y CSS para realizar operaciones matemáticas básicas.

## Descripción
Esta calculadora web fue desarrollada en PHP y se presenta con una interfaz (HTML + CSS) estilo calculadora móvil.
Permite realizar operaciones básicas:
- Suma
- Resta
- Multiplicación
- División
- Porcentaje
- Cambio de signo (+/-)

La lógica se implementa en `index.php` utilizando variables en español y control de flujo (`if`, `elseif`), y la vista se estiliza con `css/styles.css`.
