# Semana 1 – Web personal (HTML + CSS)

## Descripción
Sitio web personal multi-página desarrollado con HTML5, CSS y Bootstrap.
Incluye secciones: Inicio, Sobre mí, Experiencia, Proyectos y Contacto.

## Estructura
- `index.html`
- `sobre-mi.html`
- `experiencia.html`
- `proyectos.html`
- `contacto.html`
- `css/styles.css`
- `assets/img/` (colocar foto `perfil.jpg`)

## Cómo abrir
Abrir `index.html` con el navegador (doble clic) y navegar con el menú.

## Nota
El formulario de contacto es demostrativo (sin backend).
